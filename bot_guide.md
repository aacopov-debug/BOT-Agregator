# 📖 Руководство пользователя ArBOT Agregator

Добро пожаловать в **ArBOT Agregator** — умную платформу для поиска работы и аналитики IT-рынка. Этот бот объединяет вакансии из 7+ источников и использует AI для ускорения вашего трудоустройства.

---

## 🚀 Основные функции

### 1. Поиск вакансий
*   **Умный поиск (`/search`)**: Поддерживает операторы `AND`, `OR`, `NOT` и фильтры по зарплате (например, `python > 200к`).
*   **Фильтрация по категориям (`/filter`)**: Быстрый просмотр вакансий для Frontend, Backend, Data Science и других направлений.
*   **Горячие вакансии (`/hot`)**: Список самых свежих предложений за последние 24 часа.
*   **Тренды (`/trends`)**: Изменения рынка за неделю.

### 2. AI-инструменты для карьеры
*   **AI-Тренажер интервью (`/interview`)**: NEW! Робот-рекрутер проведет с вами пробное собеседование по выбранной вакансии (или вашему профилю), задаст вопросы и даст подробный разбор ошибок.
*   **Загрузка резюме (`/resume`)**: Отправьте PDF или текст. Бот извлечёт навыки, уровень и зарплатные ожидания.
*   **AI-анализ**: Робот-рекрутер оценит ваши сильные и слабые стороны, даст советы по карьере и предложит подходящие вакансии.
*   **Matching (Подбор)**: Оценка совместимости каждой вакансии с вашим резюме в процентах (0–100%).
*   **AI Сопроводительное письмо**: Генерация персонализированного письма под конкретную вакансию прямо из карточки.
*   **Экспорт в HTML**: Возможность скачать ваше распарсенное резюме в виде чистого HTML для печати или сохранения в PDF.

### 3. Автоматизация и уведомления
*   **Умные подписки (`/subs`)**: Подпишитесь на любой поисковый запрос, и бот сам пришлет уведомление.
*   **Режимы уведомлений**: 
    *   ⚡ **Мгновенно:** (только для Premium) Как только вакансия найдена.
    *   🕐 **Раз в час:** Бесплатный режим для всех.
    *   🌅 **Утренний дайджест:** Сводка в 9:00.
*   **Черный список (`/blacklist`)**: Скрывайте вакансии от нежелательных компаний.

### 4. Для работодателей и HR (`/hr`)
*   **Прямое размещение**: Опубликуйте вакансию, которая получит статус 🔥 **Прямой работодатель**.
*   **Продвижение**: Сделайте вакансию «Горячей», чтобы она всегда была в топе поиска и рассылок.
*   **Пакеты вакансий**: Экономьте до 30%, покупая пакеты по 3 или 10 размещений.

### 5. Личный кабинет и Бонусы
*   💳 **Баланс (`/balance`)**: Управляйте своими AI-кредитами.
*   🔄 **Daily Streak**: Заходите в бота 3 дня подряд и получайте бонусный AI-кредит бесплатно!
*   🎁 **Реферальная программа (`/referral`)**: 
    *   Пригласите 3 друзей — получите **1 день Premium** бесплатно!
    *   За каждого друга начисляется **+3 AI-кредита**.
